# Python + MySQL Practice

This repository contains my SQL revision practice done using Python's mysql-connector library.

## Features / Topics Covered
- Creating database and tables
- Inserting records
- Updating records
- Deleting records
- Selecting data with conditions
- Aggregate functions (MIN, MAX, SUM, AVG)
- ORDER BY and GROUP BY queries

## How to Run
1. Install dependencies:
   pip install mysql-connector-python

2. Create a MySQL database:
   CREATE DATABASE student_db;

3. Create the student table:
   CREATE TABLE students (id INT PRIMARY KEY, name VARCHAR(50), marks INT);

4. Run the Python file:
   python main.py

## Purpose
This repo is for practicing Python-MySQL integration and revising SQL concepts learned earlier.
